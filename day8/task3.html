PROGRAMMING LANGUAGES

  • Web development is the broad process of designing,building, and maintaining websites and web applications that are accessible via the internet,encompassing everything from creating simple static pages to complex dynamic applications. It involves a combination of skills,including web design,web programming (both front-end and back-end), client-side and server-side scripting, and database management,all working together to ensure a functional, user-friendly,and accessible online experience.

    1. HTML
    2. CSS
    3. JAVASCRIPT

  • HTML:

    Hypertext Markup Language (HTML) is the standard markup language[a] for documents designed to be displayed in a web browser. It defines the content and structure of web content.


  • CSS:

    CSS, or Cascading Style Sheets, is a stylesheet language used to describe the presentation of a document written in a markup language, most commonly HTML. It is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.


  • JAVASCRIPT:

    JavaScript (JS) is a high-level, interpreted (or just-in-time compiled) programming language that is a core technology of the World Wide Web, alongside HTML and CSS. It enables interactive and dynamic content on websites and is supported by all modern web browsers, which have dedicated JavaScript engines to execute the code.
